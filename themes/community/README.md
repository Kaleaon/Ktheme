# Community Themes

This directory contains themes contributed by the Ktheme community.

## Submitting Your Theme

1. Create your theme using the [Theme Creator](../../theme-creator)
2. Export it to JSON
3. Add your theme file here
4. Add a screenshot to the `screenshots/` folder
5. Update this README with your theme information
6. Submit a Pull Request

## Available Community Themes

### Example Theme Format

**Theme Name**: Your Theme Name  
**Author**: Your Name  
**Description**: A brief description of your theme  
**Tags**: tag1, tag2, tag3  
**File**: `your-theme-name.json`  
**Screenshot**: ![Preview](screenshots/your-theme-name.png)
